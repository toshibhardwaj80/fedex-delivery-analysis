# 📦 FedEx Supply Chain Delivery Analysis

## 📌 Objective
The objective of this project is to perform Exploratory Data Analysis (EDA) on supply chain delivery data to identify patterns related to shipment modes, cost, vendor performance, and delivery efficiency.

## 🛠 Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📊 Steps Performed
- Data Loading
- Data Cleaning (handling missing values & duplicates)
- Data Exploration (info, describe)
- Shipment Mode Analysis
- Vendor Analysis
- Country-wise Analysis
- Data Visualization (Bar Chart, Pie Chart, Histogram, Boxplot)

## 📈 Key Insights
- Air shipment mode has higher average cost compared to other shipment modes
- Vendor costs vary significantly across different suppliers
- Shipment mode plays a crucial role in determining overall delivery cost
- Certain countries show higher delivery expenses
- Most shipments are done using specific shipment modes

## 🚀 Conclusion
This analysis shows that shipment mode and vendor selection significantly impact cost and efficiency. Optimizing these factors can help improve supply chain performance.
